Detailed Description

Class for serving hovering on a cable. When the hover on cable action is called, the drone will start hovering on the cable, and the action will succeed immediately. The maneuver scheduler will then keep hovering on the cable until a new maneuver is scheduled or the timeout elapses by the duration given in the hover action call. Exposes external methods for getting the reference and updating the hover reference parameters. Other maneuver server will use this server to generate references before a new maneuver is scheduled. Will fail if the drone is not on a cable.
